# Network Troubleshooting Lab

## 📖 Description
This project focuses on basic network troubleshooting using Windows commands. The goal was to test connectivity and verify correct IP configuration on a network.

---

## ⚙️ Tools Used
- Windows Command Prompt
- Networking commands (ipconfig, ping)

---

## 🛠️ Steps Performed
1. Opened Command Prompt
2. Used the `ipconfig` command to check IP address
3. Verified:
   - IP Address
   - Subnet Mask
   - Default Gateway
4. Identified if the device was correctly connected to the network

---

## ▶️ Usage
- Open Command Prompt
- Type:ipconfig

- Analyze the network details displayed
