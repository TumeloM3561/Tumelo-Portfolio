# Linux Remote Access (SSH)

## 📖 Description
This project demonstrates how to configure secure remote access on a Linux system using SSH (Secure Shell). SSH allows a user to connect to another computer securely over a network.

---

## ⚙️ Tools Used
- Linux (Ubuntu or similar)
- Terminal
- SSH (Secure Shell)

---

## 🛠️ Steps Performed

### 1. Install SSH Server
- Enter password to access remote system
